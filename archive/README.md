# Preflight

**Creating content is free. Checking if it's good costs $100. We made it cost 10 cents.**

Preflight is a pre-publication quality check for written content. Paste your draft, get a 6-dimension quality dashboard in 3 seconds. Perspectives, evidence, logic, framing, stakeholders, manipulation — checked at once, before you publish.

It's a map, not a rulebook. Shows you the roads. You choose where to drive.

Spell-check killed typos by catching them while you write. We kill blind spots the same way.
